# 🌐 Django Portfolio Website

A modern **Full-Stack Portfolio Website** built using **Django**, **PostgreSQL**, and **Bootstrap**, designed to showcase projects, skills, and contact functionality with secure authentication and email integration.

This portfolio demonstrates backend development, database integration, REST APIs, and production-ready project structure.

---

## 🚀 Features

✅ User Authentication (Signup / Login / Logout)  
✅ Dynamic Portfolio Homepage  
✅ Projects Management System (CRUD)  
✅ Contact Form with Email Sending (SMTP)  
✅ Admin Dashboard Management  
✅ PostgreSQL Database Integration  
✅ REST API using Django REST Framework  
✅ Secure Credentials using `.env`  
✅ Responsive UI Design  
✅ Animated Background & Modern Styling
